## 七、技术实现建议

### 数据库表结构（新增）
```sql
-- 会员订阅表
CREATE TABLE subscriptions (
  id UUID DEFAULT gen_random_id() PRIMARY KEY,
  user_id UUID REFERENCES profiles(id),
  plan_type TEXT CHECK (plan_type IN ('free','monthly','yearly','lifetime')),
  status TEXT CHECK (status IN ('active','cancelled','expired')),
  started_at TIMESTAMP DEFAULT NOW(),
  expires_at TIMESTAMP,
  is_early_bird BOOLEAN DEFAULT FALSE,
  payment_id TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 生成次数记录表
CREATE TABLE generations (
  id UUID DEFAULT gen_random_id() PRIMARY KEY,
  user_id UUID REFERENCES profiles(id),
  image_url TEXT,
  resolution TEXT,
  has_watermark BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- 促销活动表
CREATE TABLE promotions (
  id UUID DEFAULT gen_random_id() PRIMARY KEY,
  code TEXT UNIQUE,
  discount_type TEXT CHECK (discount_type IN ('percentage','fixed')),
  discount_value INTEGER,
  max_uses INTEGER,
  used_count INTEGER DEFAULT 0,
  valid_until TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### 会员权益判断逻辑（前端）
```typescript
// 获取用户当前会员等级
const getUserPlan = (subscription: Subscription | null) => {
  if (!subscription || subscription.status !== 'active') return 'free';
  return subscription.plan_type; // 'monthly' | 'yearly' | 'lifetime'
};

// 检查生成次数限制
const canGenerate = (user: Profile, countToday: number, countMonth: number) => {
  const plan = getUserPlan(user.subscription);
  switch (plan) {
    case 'free': return countToday < 10;
    case 'monthly': return countMonth < 100;
    case 'yearly': 
    case 'lifetime': return true; // 无限
    default: return countToday < 10;
  }
};
```
